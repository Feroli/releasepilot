## Working Assumptions

- Implementation stack: Java 21, Spring Boot 3, Gradle.
- Infrastructure: PostgreSQL for persistence and RabbitMQ for async domain events.
- Delivery target: a repository that runs locally with Docker Compose, includes README examples for every command, preserves the AI working record, and explains the architecture decisions.
